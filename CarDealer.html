CarDealer

interface CarDealer

This is the interface for dealing with car dealers.

Public Methods

closeTransaction

closeTransaction(trans: Ts.Transaction, counteroffer: (...t: Ts.Transaction) => void) → {trans is Ts.Transaction}

Close transaction. If you respond to the counteroffer, then it won't be closed.

initiateTransaction

initiateTransaction(x: object) → {typeof Ts.Transaction}

Initiate a transaction/negotiation with a car dealer

offerAsk

offerAsk(on: Ts.Transaction, counteroffer: (t: Car) => void, test: { [testProperty in keyof keyof] }: number) → {boolean}

Offer a price to sell the car.

NOTE: This is invalid behaviour if you are the buyer in the transaction.

offerBid

offerBid(on: { t : Ts.Transaction, [id: string]: Ts.Transaction[] }, counteroffer: Car | Vector) → {boolean}

Offer a price to buy the car.

NOTE: This is invalid behaviour if you are the seller in the transaction.
